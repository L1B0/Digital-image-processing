## 0x02 imageTrans.py（第八周作业-图像变换）

### 1. 环境说明

- **运行环境**：Linux(deepin)，windows应该也行


- **语言**：Python2


- **依赖库**：cv2, numpy, pywt, matplotlib

### 2. 使用说明

运行脚本，`python imageTrans.py `即可

然后输入图像的路径（绝对路径和相对路径均可）

> 在我的环境里，cv2的dct函数处理图像时若图像的长度或高度为奇数会异常退出
